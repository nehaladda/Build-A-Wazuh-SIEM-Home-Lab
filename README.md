 # Build-A-Wazuh-SIEM-Home-Lab
A hands-on Wazuh SIEM deployment project demonstrating endpoint agent connectivity and File Integrity Monitoring (FIM) for security analysis.
Here is a one-line description and a structured README for your GitHub repository:
Overview :
This project showcases the deployment of an open-source Wazuh SIEM/XDR environment. It involves setting up a centralized security manager on an Ubuntu virtual machine and connecting a Windows host as an agent to monitor endpoint security

Key Features :
Centralized Log Management:Deployment of a Wazuh Manager to collect and analyze telemetry from endpoints. 
File Integrity Monitoring (FIM): Real-time detection and logging of file creation and deletion events to identify unauthorized system modifications.  
Troubleshooting Practice: A core component of this project focused on persistent troubleshooting to overcome configuration challenges, a critical skill for any SOC analyst

Technologies Used :
SIEM: Wazuh (Manager & Agent).  
Operating Systems: Ubuntu (Manager)
Windows (Agent).  Virtualization: Oracle VM VirtualBox

Conclusion :This lab provided practical experience in setting up security infrastructure and monitoring system integrity. It highlighted the importance of persistent troubleshooting in real-world technical environments, bridging the gap between security theory and operational practice.
